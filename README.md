# TP-FINAL-FRONT

## Requisitos
- Node.js instalado
- npm incluido con Node.js

## Pagina web

URL: https://tp-final-front-pink.vercel.app/

## Instalación y ejecución

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/tizianoluziramos/TP-FINAL-FRONT.git

2. Abrir una ventana de CMD en esa carpeta (o continuar en la misma terminal):
    ```bash
    cd TP-FINAL-FRONT

3. Instalar dependencias:

```
npm install
```
Ejecutar el entorno de desarrollo:

```
npm run dev
```
